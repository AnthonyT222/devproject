# AT Software Engineering Learning Website

A personal learning journey website showcasing my progress in software engineering and web development.

## About

This project documents my journey learning to code and exploring new programming languages and technologies. I'm using Codecademy as my primary learning platform to build the skills needed for modern software development.

### Programming Languages & Technologies
- **HTML** - Building semantic web structure
- **CSS** - Styling and layout design
- **JavaScript** - Interactive web functionality
- **TypeScript** - Type-safe JavaScript development
- **React** - Modern frontend framework
- **Node.js** - Server-side JavaScript development

### Middleware & Cloud Technologies

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/AnthonyT222/devproject.git
   ```

2. Navigate to the project directory:
   ```bash
   cd devproject
   ```

3. Open `index.html` in your web browser to view the website.

## Features

- Clean, responsive design using CSS
- Organized learning goals and objectives
- External links to learning resources
- Semantic HTML structure

## Development

This is a static website project using vanilla HTML and CSS. No build process or dependencies are required.

To make changes:
1. Edit the HTML files for content updates
2. Modify CSS files in `resources/css/` for styling changes
3. Test changes by refreshing your browser
